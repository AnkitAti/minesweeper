minesweeper
===========

A simple game of windows in Java

Mine Sweeper, a very famous game of Windows Operating System (included till Windows 7, not included in Windows 8 but available in windows store) is designed and coded in C#.Net  as we see it in Windows. So this minesweeper game is coded in Java (SE 1.7) for plattform independancy. The GUI is desinged using swing with light weight components. Since the GUI part is entirely coded in Eclipse (not in Netbeans, no DRAG AND DROP feature) so its quite simple.
In windows, the application takes the same amount of memory as windows version of minesweeper takes. So its affordable to run in your system if you can afford windows 7 or windows vista.
